MongoDB Commands Cheat Sheet

  • show dbs List all databases
  • use <database_name> Switch to a specific database or create it if it doesn't exist
  • show collections List all collections within the current database
  • db.createCollection("name") Explicitly create a new collection
  • db.collection.insertOne({ key: "value" }) Insert a single document into the collection
  • db.collection.insertMany([{ key: "v1" }, { key: "v2" }]) Insert multiple documents at once
  • db.collection.find() Retrieve all documents in the collection
  • db.collection.findOne({ key: "value" }) Retrieve a single document matching the query
  • db.collection.updateOne({ key: "old" }, { $set: { key: "new" } }) Update the first document that matches the filter
  • db.collection.deleteOne({ key: "value" }) Delete the first document that matches the filter
  • db.collection.drop() Drop (delete) the collection
  • db.dropDatabase() Drop the current database
